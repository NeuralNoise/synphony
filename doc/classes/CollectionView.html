    Class: CollectionView

    Defined in: src/client/view/common/collection.coffee
    Inherits: CompositeView

    Overview

    A view that can render a collection of models with a specific view. Initially the element is empty until render is called then once render is called, any add/remove/change events on the collection will rerender. Idea from Backbone.Marionette
