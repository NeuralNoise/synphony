    Class: BaseInteractor Abstract

    Defined in: src/client/interactor/base.coffee

    Overview

    This class is abstract.

    Base superclass for all SynPhony interactors. An interactor is a composable object that implements use cases directly. It should have at least one method named run, be named for the use case, and be asynchronous. Every effort should be made to make sure it is composable. It may also emit events that other interactors can listen to.

    See also:

    • Backbone.Events

    Direct Known Subclasses

    CirriculumDesigner, KnownFocusSearch, Segmenter, WordSearcher

    - (void) delegateEvent(eventProducer, eventName, newEventName = eventName)

    Helper function to delgate model events to the interactor for the view layer to listen to.

    Parameters:

    • eventProducer (Backbone.Events) — the producer of the event to delegate
    • eventName (String) — the event name
    • newEventName (String) — optional different name

    - (void) destroy()

    Called to clean up cyclical references between the interactor and the model layer. Also removes any event listeners.
