    Class: BaseView

    Defined in: src/client/view/common/base.coffee
    Inherits: Backbone.View

    Overview

    Base view with some extensions that make sense. Most stolen from Backbone.Marionette.

    Subclasses have a easy way to destroy which will clean up the DOM and events.

    Also there's triggers which can be used to forward DOM events to anyone listening to this view.

    Direct Known Subclasses

    CompositeView, Hardpoint, HiderView, TemplateView, SentenceListView, WordListView

    Instance Method Summary

    • - (void) destroy() Destroy the view taking care to remove event handlers so it will be garbage collected properly, as well as removing it from the DOM.

    Instance Method Details

    - (void) destroy()

    Destroy the view taking care to remove event handlers so it will be garbage collected properly, as well as removing it from the DOM. A destroy event is triggered immediately prior to removing all listeners.
