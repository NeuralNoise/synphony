    Class: KnownFocusSearch

    Defined in: src/client/interactor/known_focus_search.coffee
    Inherits: BaseInteractor

    Overview

    Strategy for searching through words or sentences for matches with containing known GPCs and focus GPCs. A known GPC is a list of GPCs that matching items MAY contain. A focus GPC is a list of GPCs that matching items MUST contain.

    The Item refered to below is any object with a .gpcs() method that returns an Array of GPC instances. So this will work on Word or Sentence instances, and in the future perhaps Story instances too.

    Instance Method Details

    - (Array<Item>) getKnownFocusItems(knownGPCs, focusGPCs, list = @models)

    Get the list of items that have the known and focus GPCs. This algorithm works on any item with a gpcs() method that returns an array of gpcs. The result is a subset of the list which contains no unknown gpcs, and contains any focus gpcs.

    Parameters:

    • knownGPCs (Array<GPC>) — array of known GPC instances
    • focusGPCs (Array<GPC>) — array of focus GPC instances
    • list (Array) — array of items with .gpcs() method (optional - defaults to collection's models)

    Returns:

    • (Array) — list of items found
